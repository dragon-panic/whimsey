Full Spanish translation of the Whimsey Tarot site.

## Approach
Custom lightweight i18n — no library. Uses App Router `[locale]` dynamic segment,
React Context for locale, and static imports for both language bundles.

## Key decisions
- No new dependencies
- URL is source of truth (`/en/...`, `/es/...`); localStorage only for root redirect
- Original `utils/` files remain untouched as English source of truth
- Spanish translations live in `i18n/content/es/`
- Hash navigation uses English data keys (locale-independent)
- Both language bundles in every page (~140KB total, fine for static site)

## File structure
```
i18n/config.js              → locales array, defaultLocale
i18n/useTranslations.js     → useT() hook for UI strings
i18n/useContent.js          → useCards(), useAstrology(), usePlanets(), useSigns(), useCourt()
i18n/ui/{en,es}.json        → UI chrome strings
i18n/content/es/*.js        → Spanish content (mirrors utils/ translatable exports)
app/layout.js               → minimal root layout (<html><body>)
app/page.js                 → root redirect (localStorage check → /en/ or /es/)
app/[locale]/layout.js      → LocaleProvider + generateStaticParams
app/[locale]/{page,court,signs,planets}/page.js
app/components/LanguageSwitcher.js → EN/ES toggle
```
