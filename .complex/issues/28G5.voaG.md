Update all components to use the i18n hooks instead of direct imports and hardcoded English strings.

Depends on: ROuH (hooks exist), OtRC (en.json exists), k5Ih (pages are under [locale]).

See subtasks for per-component details. General pattern for each page:
1. Replace direct `utils/` imports of translatable data with `useContent` hooks
2. Keep direct imports of non-translatable data (symbols, element maps, date ranges)
3. Replace inline English strings with `useT()` lookups
4. Prefix cross-page `<Link href>` with `/${locale}/`
5. Get locale via `useLocale()` from context
