Create `app/components/LanguageSwitcher.js`.

## Behavior
- Reads current locale from `useLocale()`
- Reads current pathname from `usePathname()`
- On click: replaces locale segment in path, preserves hash, saves to localStorage, navigates
- Renders EN / ES buttons reusing the existing `deck-switcher` / `deck-button` CSS classes

## Example
```js
const switchLocale = (newLocale) => {
  const pathWithoutLocale = pathname.replace("/${currentLocale}", '') || '/';
  const hash = window.location.hash;
  localStorage.setItem('whimsey-locale', newLocale);
  router.push("/${newLocale}${pathWithoutLocale}${hash}");
};
```

Add this component to Navigation.js alongside DeckSwitcher.
